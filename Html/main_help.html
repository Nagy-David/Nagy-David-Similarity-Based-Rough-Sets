Similarity Based Rough Sets

Similarity Based Rough Sets is a partial generalized Pawlakian approximation space which is based on a tolerance relation. This relation represents the similarity of objects. The system of base sets is generated by the correlation clustering which is a clustering technique based on a tolerance relation. The base sets contains objects that are similar to each other.

Software description

The software is capable of creating the aforementioned approximation space and in this space it can approximate arbitrary sets. With the approximation we can reveal hidden patterns and information among the data. There are two possible ways for the user to give the input data set:

  1. Generating random two-dimensional points
  2. Read a dataset from a file

After getting the input points the user can use the following possibilites to control the software:

Clicking on the Generate button the given number of random points will be generated. Naturally this button cannot be used if previously the user read the dataset from a file. Clicking on the Run button a search algorithm (can be downloaded using this link) is executed which finds a quasi optimal partition. Singleton clusters represent very little information, because the system could not consider its member similar to any other objects without increasing the value of the cost function of the correlation clustering. Because of this, the software discards them and creates the system of base sets of without them. After defining the base sets, the user can select a set of points for approximation by clicking on the Approximate button. The points to be approximated can be given as an interval in the following format: start_id-end_id. With Display Approximation button the user can check the result of the approximation. The set to be approximated is displayed with green points, the lower approximation contains red points and the upper apporixmation is displayed with blue points.

Toolbars

With the buttons on the toolbar, the user can control how the data points are displayed in the software.
  1. Display points in black
  2. Display clusters
  3. Display representatives (with rectangles)
  4. Display point IDs

Annotation

In the software, the user has the option to insert the members of the left-out singleton clusters to any base set. Natrually, two singleton clusters cannot be merged together. The software offers some guidelines to help the user in the decision. The recommended base set is the one whose representative member is the closest to the member of the given singleton.
