Generating Random Points

  • In this option, the dataset consists of random two-dimensional points in a plane.
  • The base of the tolerance relation is the Euclidean distance of the objects.
  • The user can use the following form to give the possible inputs.
  • The base of the tolerance relation is the Euclidean distance of the objects.
  • The user can set two threshold values: one for the similarity and one for the dissimilarity
  • The tolerance relation is defined in the following way:
    • If two objects are closer than the similarity threshold, then they are similar
    • If two objects are closer than the dissimilarity threshold, then they are dissimilar
    • They are neutral otherwise
